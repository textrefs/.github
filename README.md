## What TextRefs is

- an open, persistent, machine-readable registry of canonical text references;
- a set of open standards, JSON-LD shapes, and reference implementations;
- a curated source of mappings between canonical references and external identifiers (CTS URNs, Wikidata IDs, DOIs, ARKs, Perseus URLs, etc.).

## What TextRefs is _not_

- a full-text database or a critical edition;
- a publisher or commercial SaaS;
- a substitute for Perseus, Loeb, TLG, PHI, DTS, CTS, or any library catalogue.

## Status

Pre-1.0, bootstrap-funded. TextRefs is being set up as a Zürich-based non-profit association (_Verein_ in formation), seeking tax-exempt status. Per [ADR-0001](https://github.com/textrefs/textrefs.org/blob/main/decisions/ADR-0001-standard-first-engine-deferred.md), the standard is published first; the registry/resolver engine is intentionally deferred and will be rebuilt against the fixed standard.
